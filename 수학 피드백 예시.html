⭐️ 학습 분석 리포트

우수한 부분

'다면체'의 기본 정의, '각기둥의 부피', '원기둥의 부피' 등 여러 핵심 개념에서 100%의 성취도를 보였습니다. 특히 기둥의 부피를 구하는 공식을 정확히 알고 적용하고 있습니다.

보완이 필요한 부분

'구의 겉넓이와 부피' 단원에서 개념 이해도가 매우 낮습니다.

  • '구의 부피' (성취도 0.0%): 반지름 r인 구의 부피 공식 V = (4/3)πr³을 이용하여 값을 계산하는 데 어려움을 겪고 있습니다.
  • '구의 겉넓이' (성취도 13.0%): 구의 겉넓이 공식 S = 4πr²을 적용하지 못하고 있으며, 특히 '반구'의 겉넓이를 구할 때 (곡면 넓이) + (밑면 넓이)를 함께 계산하는 응용력이 부족합니다.

⏰ 5분 집중 보완 학습 자료

[핵심 개념/공식 다시 보기]

반지름의 길이가 r인 구가 있을 때, 겉넓이와 부피 공식은 다음과 같습니다. 이 공식들은 반드시 암기해야 합니다.

  • 구의 겉넓이 (S)
    구의 겉넓이는 반지름이 r인 원의 넓이(πr²)의 4배와 같습니다.
    S = 4πr²
  • 구의 부피 (V)
    V = (4/3)πr³

(주의!) 겉넓이는 제곱(r²)이고, 부피는 세제곱(r³)입니다.

[응용] 반구의 겉넓이와 부피

  • 반구의 부피: 구의 부피를 단순히 반(1/2)으로 나누면 됩니다.
    V = (1/2) × (4/3)πr³ = (2/3)πr³
  • 반구의 겉넓이: 구의 겉면 절반(2πr²)과 잘린 밑면(원의 넓이, πr²)을 더해야 합니다.
    S = (1/2 × 4πr²) + (πr²) = 2πr² + πr² = 3πr²

✏️ 5분 보완 연습 문제

  1. 반지름의 길이가 3cm인 구의 겉넓이를 구하시오.
  2. 반지름의 길이가 3cm인 구의 부피를 구하시오.
  3. 반지름의 길이가 6cm인 반구의 겉넓이를 구하시오.
  4. 지름의 길이가 6cm인 반구의 부피를 구하시오.

🔑 정답 및 해설

  1. 36π cm²

    (겉넓이 공식) S = 4πr²
    S = 4 × π × 3² = 4 × π × 9 = 36π cm²

  2. 36π cm³

    (부피 공식) V = (4/3)πr³
    V = (4/3) × π × 3³ = (4/3) × π × 27 = 4 × π × 9 = 36π cm³

  3. 108π cm²

    (반구 겉넓이 공식) S = 3πr² (또는 (구 겉넓이 절반) + (밑면 넓이))
    S = 3 × π × 6² = 3 × π × 36 = 108π cm²

  4. 18π cm³

    지름이 6cm이므로, 반지름 r = 3cm 입니다.
    (반구 부피 공식) V = (2/3)πr³
    V = (2/3) × π × 3³ = (2/3) × π × 27 = 2 × π × 9 = 18π cm³

⭐️ 학습 분석 리포트

우수한 부분

'집합의 뜻'을 명확히 구분하고 '유한집합의 원소의 개수(n(A))'를 세는 등 집합의 기본 개념은 잘 정립되어 있습니다.

보완이 필요한 부분

'기호 ∈, ⊂의 사용' 및 '집합을 원소로 가지는 집합'의 성취도가 저조합니다. 이는 '원소(element)'와 '부분집합(subset)'의 개념을 혼동하고 있음을 의미합니다.

특히, A = {∅, {1}, 2} 와 같이 집합 기호(∅, { }) 자체를 원소로 포함하는 복잡한 집합이 주어졌을 때, 기호(∈)와 포함 관계(⊂)를 정확히 구별하는 데 어려움을 겪고 있습니다.

⏰ 5분 집중 보완 학습 자료

[핵심 개념/공식 다시 보기]

  • 원소 기호 (∈)
    • (원소) ∈ (집합) 의 형태로 사용합니다.
    • 기호 왼쪽(A)이 기호 오른쪽(B) 집합의 { ... } 괄호 안에 콤마(,)로 구분된 '멤버'로 그대로 들어있는지 확인합니다.
    • 예: A = {1, 2, 3} 일 때, 1 ∈ A 입니다.
  • 부분집합 기호 (⊂)
    • (집합) ⊂ (집합) 의 형태로 사용합니다.
    • 기호 왼쪽(C) 집합의 { ... } 괄호 안에 있는 모든 원소가 기호 오른쪽(D) 집합의 원소로 포함되는지 확인합니다.
    • 예: A = {1, 2, 3} 일 때, 1은 A의 원소이므로, 이 원소로 새로운 집합을 만든 {1}은 A의 부분집합입니다. 즉, {1} ⊂ A 입니다.

[가장 중요!] 헷갈리는 예제 분석

집합 A = {∅, 1, {2}} 를 기준으로 분석해 봅니다.

이 집합 A의 '원소'는 ∅ (기호), 1 (숫자), {2} (집합) 이렇게 3개입니다.

  • 원소(∈) 판단 (모양 그대로 찾기)
    • ∅ ∈ A (참) : A의 원소 목록에 ∅ 모양이 있습니다.
    • 1 ∈ A (참) : A의 원소 목록에 1이 있습니다.
    • {2} ∈ A (참) : A의 원소 목록에 {2} 모양이 있습니다.
    • 2 ∈ A (거짓) : A의 원소 목록에 2라는 숫자는 없습니다.
  • 부분집합(⊂) 판단 (원소를 뽑아 집합 만들기)
    • ∅ ⊂ A (참) : 공집합(∅)은 모든 집합의 부분집합입니다. (규칙)
    • {1} ⊂ A (참) : A의 원소 1을 뽑아서 집합으로 만들었습니다.
    • {{2}} ⊂ A (참) : A의 원소 {2}를 뽑아서 집합으로 만들었습니다.
    • {{2}, 1} ⊂ A (참) : A의 원소 {2}와 1을 뽑아서 집합으로 만들었습니다.

✏️ 5분 보완 연습 문제

집합 A = { a, {a}, {a, b} } 일 때, 다음 보기 중 옳은 것(T)과 옳지 않은 것(F)을 판별하시오.

  1. a ∈ A
  2. {a} ∈ A
  3. {a} ⊂ A
  4. b ∈ A
  5. {a, b} ∈ A
  6. {a, b} ⊂ A
  7. {{a}} ⊂ A

🔑 정답 및 해설

  1. T (참)

    (해설) 집합 A의 원소 { a, {a}, {a, b} } 목록에 a가 그대로 포함되어 있습니다.

  2. T (참)

    (해설) 집합 A의 원소 목록에 {a}가 그대로 포함되어 있습니다. (이때 {a}는 원소로 취급됩니다.)

  3. T (참)

    (해설) 집합 A의 원소 a를 뽑아서 새로운 집합 {a}를 만들 수 있으므로, {a}는 A의 부분집합입니다. (2번의 원소 {a}와는 다른 이유로 참입니다.)

  4. F (거짓)

    (해설) A의 원소 목록에 b는 없습니다. {a, b}라는 집합 안에 b가 있지만, b 자체가 A의 원소는 아닙니다.

  5. T (참)

    (해설) A의 원소 목록에 {a, b}라는 집합이 그대로 포함되어 있습니다.

  6. F (거짓)

    (해설) {a, b}가 A의 부분집합이 되려면, 원소 a와 원소 b가 모두 A의 원소여야 합니다. 하지만 4번에서 봤듯이 b는 A의 원소가 아니므로, {a, b}는 A의 부분집합이 될 수 없습니다.

  7. T (참)

    (해설) 집합 A의 원소 {a}를 뽑아서, 이것으로 새로운 집합 {{a}}를 만들 수 있으므로, {{a}}는 A의 부분집합입니다.

예시 3: 수학 2 (도함수 그래프 해석)

⭐️ 분석 리포트 ⏰ 보완 자료 ✏️ 연습 문제 🔑 정답 및 해설

⭐️ 학습 분석 리포트

우수한 부분

미분법 공식을 이용한 '도함수의 계산'이나 '접선의 기울기'를 구하는 연산 능력은 우수합니다.

보완이 필요한 부분

'함수 y=f'(x)의 그래프의 해석' 성취도가 저조합니다. 이는 도함수 y=f'(x)의 그래프가 주어졌을 때, 원래 함수 y=f(x)의 증가·감소 상태와 극점을 추론하는 데 어려움을 겪고 있음을 의미합니다.

특히, f'(x) 그래프의 기울기가 아닌, f'(x)의 함숫값(y값)의 부호(+, -)가 f(x)의 증가·감소를 결정한다는 핵심 관계를 혼동하고 있습니다.

⏰ 5분 집중 보완 학습 자료

[핵심 개념/공식 다시 보기]

도함수 f'(x)의 그래프를 해석하여 원래 함수 f(x)의 그래프 개형을 그릴 수 있습니다. 가장 중요한 것은 f'(x)의 부호입니다.

f'(x)의 부호 → f(x)의 증가/감소

  • f'(x) > 0 인 구간 ( f'(x) 그래프가 x축보다 위에 있는 구간)
    → f(x)는 증가(↑)합니다.
  • f'(x) < 0 인 구간 ( f'(x) 그래프가 x축보다 아래에 있는 구간)
    → f(x)는 감소(↓)합니다.

f'(x)의 부호 변화 → f(x)의 극점

f'(x) = 0 이 되는 지점(f'(x)가 x축과 만나는 지점)의 좌우에서 부호가 바뀌면, f(x)는 극값을 가집니다.

  • (극대): f'(x)의 부호가 양(+)에서 음(-)으로 바뀔 때 (증가↑ → 감소↓)
  • (극소): f'(x)의 부호가 음(-)에서 양(+)으로 바뀔 때 (감소↓ → 증가↑)

(주의!) f'(x) 그래프의 기울기가 증가/감소하는 것(즉, f'(x)의 증감)은 f(x)의 오목/볼록(변곡점)과 관련되며, f(x)의 증가/감소와는 직접적인 관계가 없습니다.

✏️ 5분 개념 적용 예제

(규칙 (B) 적용: 이미지가 필수적인 KC이므로, 그래프를 텍스트로 묘사한 '예제'를 제공합니다.)

예제 1)

삼차함수 f(x)의 도함수인 y=f'(x)의 그래프가 아래로 볼록한 이차함수(포물선)이며, x축과 x = -1 , x = 2 에서 만난다고 가정합니다. f(x)의 증가/감소와 극점을 판단하시오.

예제 2)

어떤 함수 f(x)의 도함수인 y=f'(x)의 그래프가 x축과 x = 0 , x = 3 , x = 5 에서 만난다고 가정합니다.

  • x < 0 에서는 f'(x) < 0 (x축 아래)
  • 0 < x < 3 에서는 f'(x) > 0 (x축 위)
  • 3 < x < 5 에서는 f'(x) > 0 (x축 위)
  • x > 5 에서는 f'(x) < 0 (x축 아래)

f(x)의 증가/감소와 극점을 판단하시오.

🔑 정답 및 해설

[예제 1 상세 풀이 과정]

  1. f'(x)의 부호 분석: y=f'(x)는 x=-1과 x=2에서 x축과 만납니다. 그래프가 아래로 볼록하므로,
    • x < -1 일 때, f'(x) > 0 (그래프가 x축 위에 있음)
    • -1 < x < 2 일 때, f'(x) < 0 (그래프가 x축 아래에 있음)
    • x > 2 일 때, f'(x) > 0 (그래프가 x축 위에 있음)
  2. f(x)의 증감 분석: f'(x)의 부호를 바탕으로 f(x)의 증감을 판단합니다.
    • x < -1 일 때, f(x)는 증가(↑)합니다.
    • -1 < x < 2 일 때, f(x)는 감소(↓)합니다.
    • x > 2 일 때, f(x)는 증가(↑)합니다.
  3. f(x)의 극점 판단:
    • x = -1 에서 f'(x)의 부호가 양(+) → 음(-)로 바뀌므로, f(x)는 극댓값을 가집니다.
    • x = 2 에서 f'(x)의 부호가 음(-) → 양(+)으로 바뀌므로, f(x)는 극솟값을 가집니다.

[예제 2 상세 풀이 과정]

  1. f(x)의 증감 분석:
    • x < 0 일 때, f'(x) < 0 이므로 f(x)는 감소(↓)
    • 0 < x < 3 일 때, f'(x) > 0 이므로 f(x)는 증가(↑)
    • 3 < x < 5 일 때, f'(x) > 0 이므로 f(x)는 증가(↑)
    • x > 5 일 때, f'(x) < 0 이므로 f(x)는 감소(↓)
  2. f(x)의 극점 판단:
    • x = 0 에서 f'(x)의 부호가 음(-) → 양(+)로 바뀌므로, f(x)는 극솟값을 가집니다.
    • x = 3 에서 f'(x)의 부호가 양(+) → 양(+)로 바뀌지 않습니다. f'(x)=0 이지만 좌우 부호가 같으므로, f(x)는 극값을 갖지 않고 잠시 멈칫했다가 계속 증가합니다.
    • x = 5 에서 f'(x)의 부호가 양(+) → 음(-)로 바뀌므로, f(x)는 극댓값을 가집니다.
